# Proyecto-Django
Tarea Final: Proyecto Web con Django

## Objetivo del Proyecto

Este proyecto es la **Tarea Final** del curso, cuyo objetivo principal fue desarrollar una aplicación web completa utilizando el framework **Django**. Se aplicó el ciclo completo de desarrollo: desde la configuración del entorno virtual, la estructuración del proyecto y la aplicación, hasta la implementación de **Model-View-Template (MVT)**.

---

##  Características Implementadas

* **Configuración MVT básica:** Correcta conexión entre la URL, la vista (`views.py`) y el template (`inicio.html`).
* **Gestión de Templates:** Uso de un template simple con inyección de contexto (`{{ titulo }}`).
* **Estilo Básico:** El template utiliza **Tailwind CSS** a través de su CDN para una presentación limpia y moderna.
* **Vistas:** Incluye la vista `inicio` y la vista opcional **"Acerca de"**.
* **Buenas Prácticas:** Repositorio configurado con un archivo `.gitignore` para excluir el entorno virtual (`.venv`) y la base de datos local (`db.sqlite3`).
